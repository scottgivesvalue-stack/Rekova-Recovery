# Rekova Recovery Academy

A React + Vite learning platform for the Rekova Recovery kit.

## What is included

- Branded hero landing experience
- Visual, audio, and text learning sections
- Recovery kit item library with usage and benefits
- Placeholder video/audio guides ready for real content
- Built with React and TypeScript for easy mobile migration

## Getting started

1. Add the local Node install to your shell environment if you haven't already:

```bash
export PATH="$HOME/.local/node/bin:$PATH"
```

2. Install dependencies:

```bash
cd "/Users/scottgivesvalue/Desktop/Rekova App"
npm install
```

3. Start the app:

```bash
npm run dev
```

4. Open the local URL shown in the terminal, usually:

```bash
http://localhost:4173/
```

## Next step

This app is ready to evolve into a mobile experience using React Native or Expo.
